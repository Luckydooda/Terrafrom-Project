# GitHub Actions Examples

This directory contains example applications that demonstrate how to use the GitHub Actions workflows.

## Example Applications

### Python Application
A simple Python application with:
- FastAPI web framework
- Unit tests with pytest
- Code coverage reporting
- Docker containerization

### Node.js Application
A Node.js application with:
- Express.js framework
- Jest testing
- ESLint configuration
- Docker support

### Docker Application
A multi-service Docker application with:
- Docker Compose configuration
- Multi-stage builds
- Health checks
- Production-ready setup

## Usage

1. Copy an example application to your repository
2. Copy the appropriate workflow file to `.github/workflows/`
3. Customize the workflow for your needs
4. Configure required secrets
5. Push to trigger the workflow
